# Simple Git Tutorial

## Sphinx Installation

1. Install [Python 3](https://www.python.org/downloads/)
2. `pip install -U Sphinx`
3. Done!

## Build

1. `make clean && make html`
2. `python -m http.server`
3. `open http://localhost:8000/_build/html/`
